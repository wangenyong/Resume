* 项目背景：公司原有数据采集平台采用Jenkins调度编排任务，涉及多台服务器之间脚本相互调用，结构复杂，运行不稳定且难以维护。

* 项目职责：负责整个项目重构的规划、实施以及维护。
* 项目结果：重构完成后，整个数据采集平台运行稳定，可维护性高，也增强了上游市场竞争分析以及财务收益等系统的稳定性。
